
I LemonLemon ligger styleguide med CSS för den här applikationen.
I LemonLemon.js sköts förutom funktionalitet för att använda själva styleguiden även funktionalitet för navigation för den här kopplade SPA-applikationen.
Det gör det möjligt för andra SPA applikationer att koppla sig till LemonLemon och använda sig av samma CSS och JS navigationssystem som den här applikationen.
I LemonLemon mappen pics ligger våra egna bilder vi använder i LemonLemons andra applikationer.
